# PlayLabs Studio — Intern Handbook

A six-week builder apprenticeship in a single page: find a real problem, build an MVP with AI, ship it, and learn when to keep going or kill it. Designed to be handed to an intern and used alongside a mentor.

**Live site:** https://reynoldharbin.github.io/PlayLabs/ *(goes live once you enable GitHub Pages — see below)*

## What's inside

- The **Builder Loop** (9 steps) and a week-by-week six-week plan
- The **2026 AI tool stack** with links — ChatGPT + Codex, GitHub, Vercel, Neon, v0, and more
- An interactive **60-second kill check** that scores an idea live (BUILD / CAREFUL / WALK-AWAY)
- A **"validate with commitment before code"** rule and keep-vs-kill criteria
- **Progress tracking** — checkboxes and the kill check persist in the browser
- Deliverables, the daily routine, and the rules

## View it

Open `index.html` in any browser — no build step, no dependencies.

To serve it locally:

```bash
python3 -m http.server 8080
# then open http://localhost:8080
```

## Host it free with GitHub Pages

1. Push this repo to GitHub.
2. **Settings → Pages → Source:** `main` branch, `/ (root)` → **Save**.
3. The handbook goes live at the URL above in about a minute.
